# pravda
Телеграм бот, который выдает тебе случайную вырезку из газеты "Правда" сегодняшней даты. Цель - выбрать верный год из нескольких предлагаемых.
```
pip install -r requirements.txt
```
После установки зависимостей, запусти бота:
```
python bot.py
```
Бот будет работать в Telegram, и ты сможешь начать игру, отправив команду /start. Он будет присылать тебе вырезки из газеты "Правда", и тебе нужно будет выбрать правильный год публикации. Удачи!
